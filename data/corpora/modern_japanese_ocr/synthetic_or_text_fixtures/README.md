# Synthetic Or Text Fixtures

This area is for generated OCR fixture resources and text-only resources.

Examples:

- Font-generated image datasets.
- Bilingual or terminology text corpora.
- Synthetic images created for cleanup/serialization tests.
- CSV/JSON text fixtures that are useful for OCR evaluation support but are not
  scanned-page OCR data.

## Commit Policy

Payloads are ignored by default. Track READMEs and manifest examples.

## Use

These resources are useful for guardrails and fixtures, but they should not be
confused with real scanned-page OCR evaluation data.
